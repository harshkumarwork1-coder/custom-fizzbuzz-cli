A Node.js command-line application that generates FizzBuzz output for a user-defined number range.

## Features
- Takes custom starting and ending numbers from user input
- Implements classic FizzBuzz logic
- Input validation for safer execution
- Handles large ranges with optional limits

## Tech Stack
- JavaScript
- Node.js
- Readline module

## How to Run

1. Clone the repository
2. Run in the wondow terminal or VA code terminal:
   node filename.js
3. Enter starting and ending numbers when prompted

## Example

Enter starting number: 10  
Enter ending number: 20
